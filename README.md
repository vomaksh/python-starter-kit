# Python Starter Kit

[![Copier](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/copier-org/copier/master/img/badge/badge-grayscale-inverted-border-orange.json)](https://github.com/copier-org/copier)

A minimal Python project template bootstrapped using [Copier](https://copier.readthedocs.io/).

This repository is intended to be used as a **starting point** for new Python projects, providing a clean baseline with modern tooling and devcontainers support out of the box.

## Features

- `Python 3.14` out of the box
- `pyproject.toml`-based configuration
- `uv`-friendly dependency management
- `ruff` for code formatting
- `pyrefly` for LSP
- `devcontainers` configured with required extensions like
  - `ruff`
  - `pyrefly`
  - `jupyter`
  - `toml`
